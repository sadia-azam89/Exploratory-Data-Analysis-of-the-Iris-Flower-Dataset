# Exploratory-Data-Analysis-of-the-Iris-Flower-Dataset
Project Overview  This project performs Exploratory Data Analysis (EDA) on the famous Iris dataset using Python, Pandas, Matplotlib, and Seaborn.  The notebook analyzes the dataset structure, generates summary statistics, and visualizes relationships between features to better understand patterns among different Iris species.
Dataset

The dataset used is the Iris dataset, which contains measurements of iris flowers from three different species:
Setosa
Versicolor
Virginica

Features:
Sepal Length (cm)
Sepal Width (cm)
Petal Length (cm)
Petal Width (cm)
Species (Target Variable)
